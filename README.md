## Description
IPTV Desktop is a cross-platform GUI application built with Qt C++. Watch free TV channels from all over the world using [IPTV API](https://github.com/iptv-org/api).  

**New in 2.0**: offline cache, favorites, M3U import, search bar, dark mode, and auto-skip broken streams.

## Features
- **Sortable lists**: Countries, Regions, Subdivisions, Languages, Categories
- **Media controls**: Play, Pause, Next, Previous, Mute, Volume slider
- **Favorites**: Save your favorite channels for quick access
- **Offline mode**: Cache channels and streams
- **Search bar**: Filter channels instantly
- **M3U import**: Add your own playlists
- **Dark/Light mode toggle**
- **Auto-skip broken streams**
- **Daily automatic updates** from IPTV API

# IPTV Desktop

A desktop IPTV player built with **Qt 5.15.2** for macOS.

---

## **Requirements**

- macOS 10.13 or later  
- **Qt 5.15.2** (clang_64)  
- Xcode command line tools (`clang++`)  

Optional: Use the SDK 10.15 for full Qt compatibility, or silence SDK warnings with `CONFIG+=sdk_no_version_check`.
